# Customized Visualization and Classification of Sales Data

This project analyzes B2B women clothing sales data using visualization and machine learning techniques.

Features
- Power BI dashboards for sales visualization
- Linear Regression model for order prediction
- Classification model for sales categorization
- State trend analysis
- Root cause analysis

Technologies Used
- Python
- Streamlit
- Power BI
- Pandas
- NumPy
- Scikit-learn
